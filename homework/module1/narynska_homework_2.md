## Завдання 1. Ієрархія каталогів Linux (1 бал)

### 1.1 Перехід у кореневий каталог `/` і перегляд вмісту
```bash
cd /
ls -la
```

### 1.2 Перехід у `/etc` і перегляд вмісту
```bash
cd /etc
ls -la
```

### 1.3 Перехід у `/home` і перегляд списку користувачів
```bash
cd /home
ls -la
```

## Завдання 2. Файли, каталоги та посилання (2 бали)

### 2.1 Створення нового каталогу в домашньому каталозі
```bash
cd ~
mkdir -p lab2
```

### 2.2 Створення файлу всередині каталогу
```bash
touch ~/lab2/file.txt
```

### 2.3 Копіювання файлу у нове ім'я
```bash
cp ~/lab2/file.txt ~/lab2/file_copy.txt
```

### 2.4 Перейменування копії
```bash
mv ~/lab2/file_copy.txt ~/lab2/file_renamed.txt
```

### 2.5 Створення жорсткого посилання
```bash
ln ~/lab2/file.txt ~/lab2/file_hardlink.txt
```

### 2.6 Створення символічного посилання
```bash
ln -s ~/lab2/file.txt ~/lab2/file_symlink.txt
```

### 2.7 Пошук файлу за іменем
```bash
find /home -name "file.txt" 2>/dev/null
```

Приклад очікуваного результату:
```text
/home/student/lab2/file.txt
```

## Завдання 3. Права доступу (1 бал)

### 3.1 Перегляд прав створеного файлу
```bash
ls -l ~/lab2/file.txt
```

### 3.2 Надання файлу прав тільки на читання
```bash
chmod 444 ~/lab2/file.txt
ls -l ~/lab2/file.txt
```

### 3.3 Надання власнику права на запис
```bash
chmod u+w ~/lab2/file.txt
ls -l ~/lab2/file.txt
```

### 3.4 Перегляд значення `umask`
```bash
umask
```

### 3.5 Встановлення нового значення `umask` (приклад `022`)
```bash
umask 022
umask
```

## Завдання 4. Користувачі (1 бал)

### 4.1 Створення нового користувача
```bash
sudo useradd -m -s /bin/bash trainee
```

### 4.2 Додавання користувача до sudo-групи
```bash
sudo usermod -aG sudo trainee
```

### 4.3 Перевірка, що користувач існує
```bash
getent passwd trainee
```

Приклад очікуваного результату:
```text
trainee:x:1001:1001::/home/trainee:/bin/bash
```
